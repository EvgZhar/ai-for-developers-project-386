# Расширение дискового пространства в VirtualBox (русскоязычный интерфейс)

## Шаг 1. Расширить виртуальный диск

1. Откройте VirtualBox (главное окно менеджера)
2. Убедитесь, что виртуальная машина **выключена**
3. Выберите меню **Файл → Инструменты → Менеджер виртуальных носителей** (или нажмите `Ctrl+D`)
4. В списке выберите ваш .vdi-диск
5. В правой части окна найдите поле **Размер** → рядом кнопка **Изменить размер** (значок шестерёнки)
6. Введите новый размер в ГБ или МБ, нажмите **Применить**

## Шаг 2. Расширить раздел внутри гостевой ОС

### Если гость — Linux

**Вариант A (GParted LiveCD):**
1. Скачайте ISO GParted Live, подключите к ВМ (Носители → Дистрибутив Live CD/DVD)
2. Загрузитесь с него
3. В графическом интерфейсе: выберите корневой раздел → **Partition → Resize/Move** → растяните до конца свободного места → **Apply**

**Вариант B (fdisk + resize2fs, через терминал):**
```bash
sudo fdisk /dev/sda
# d  (удалить раздел)
# n  (создать новый раздел)
# Укажите тот же номер, тот же First sector, для Last sector нажмите Enter (всё свободное место)
# w  (записать изменения)
sudo partprobe
sudo resize2fs /dev/sda1   # sda1 — ваш корневой раздел
```

### Если гость — Windows

1. Запустите ВМ
2. Нажмите `Win+R` → введите `diskmgmt.msc` → Enter
3. Правой кнопкой по нужному тому → **Расширить том**
4. Следуйте мастеру

**Готово.** После этого ОС увидит новое пространство.
